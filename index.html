About

FormJS is a javascript library to make form validation more easy.

Actually the library has up to 15 validation presets, but you can add your own validations if necessary.

Some of current features includes:

  • Non empty field validation
  • CPF validation
  • Formated field validation
  • Length validation
  • And much more

How to use

Into your html, place this script tag to import FormJS to your project:

<script src="https://cdn.jsdelivr.net/gh/Raffa064/FormJS@main/form.js"></script>

You can also download FormJS and add it's file to your project.

After it, you just need to instanciate FormJS and use it to create your validators:

const formJS = FormJS()

const validator = formJS()
    .notEmpty()
    .minLength(5)
    // Add other rules here

const name = 'Raffa064'

if (validator(name)) {
    // Valid name
} else {
    // Invalid name
}

Validation rules

See below a complete list of all validation rules available in FormJS:

  • notEmpty()

    This rule checks if the input value is not empty.

  • minLength(min)

    This rule checks whether the input has a minimum size, defined through a parameter.

  • maxLength(max)

    This rule checks whether the input has a maximum size, defined through a parameter.

  • rangeLength(min, max)

    This rule works as shorthand for minLength and maxLength.

  • fixedLength(...fixedLengths)

    Check if the input length matches with almost one of the lengths passed through the parameters.

  • match(...regexPatterns)

  • number()

  • range(min, max)

  • validCPF(checkSeparators)

  • format(...formatStrings)

  • contains(...term)

  • containsAll(...terms)

  • prefix(...prefixes)

  • sufix(...sufixes)

Like you see, FormJS has a lot of default validation rules that you can combine to match your requirements. But if you still need something more specific, you can create your own validation rules, like is showing bellow:

const formJS = FormJS()

const validator = formJS()
    .createRule('custom', ([param0, param1], value) => {
        var customLogic = ... // Make your own logic (boolean)
        return customLogic
    })
    .custom(param0, param1)
